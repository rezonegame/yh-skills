# Board Game Box Design

桌游包装盒的完整设计指南，涵盖尺寸、结构设计、封面布局和生产规格。

## 🤖 执行流程

### 第1步：确定包装盒类型

选择适合游戏的盒子类型：

**欧洲/国际标准尺寸**：
- Micro: 105×155×35mm（54张卡牌+规则）
- Small: 155×215×45mm（108张卡牌+指示物）
- Medium: 215×295×55mm（180张卡牌+游戏板+指示物）
- Large: 235×315×75mm（300+张卡牌+游戏板+大量指示物）
- Extra Large: 295×395×85mm（500+张卡牌+大型游戏板+组件）

**美式尺寸**：
- Small: 140×216×38mm
- Medium: 216×279×51mm
- Large: 229×305×64mm
- Extra Large: 279×356×76mm

### 第2步：选择盒型结构

**Two-Piece Box（上下盖）**：
- 最常见类型
- 易于生产
- 成本效益好

**Book-style Box（书本式）**：
- 高端外观
- 磁性闭合
- 较昂贵

**Tuck Box（插入式）**：
- 小型卡牌游戏
- 低成本
- 简单结构

### 第3步：设计封面艺术

**封面元素层级**：
1. **游戏标题** - 最醒目，顶部中央
2. **主视觉** - 游戏场景/角色
3. **组件展示** - 盒内内容物预览
4. **玩家数量/时长** - 清晰可见
5. **年龄标识** - 必要的合规要求

**尺寸指导**：
- 标题高度：占封面 15-20%
- 主视觉：占封面 40-50%
- 文字最小：12pt（30cm观看距离）

### 第4步：设计书脊和背面

**书脊设计**：
- 游戏标题（竖排或横排）
- Logo/图标
- 宽度：15-25mm（标准盒）

**背面元素**：
- 游戏简介
- 游戏特色列表
- 组件清单
- 公司信息
- 条形码区域

### 第5步：设计内衬和托盘

**内衬类型**：
- 纸板插片 - 低成本
- 塑料托盘 - 耐用
- 泡沫托盘 - 高级保护
- 无内衬 - 环保选择

**托盘设计原则**：
- 组件固定良好
- 易于取放
- 最小化空间浪费
- 考虑收纳效率

### 第6步：验证制作规格

**厚度选择**：
- 标准纸板：1.5-2mm
- 厚纸板：2-2.5mm
- 豪华：3mm+

**表面处理**：
- 哑光 - 标准选择
- 光油 - 鲜艳色彩
- UV 涂层 - 耐久保护
- 烫金/击凸 - 高级效果

**文件规格**：
- 300 DPI @ 最终尺寸
- CMYK 颜色模式
- 出血：5mm 四周
- 文件格式：PDF/X-4
