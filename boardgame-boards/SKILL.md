# Board Game Board Design

游戏板和玩家垫板的完整设计指南，涵盖布局系统、折叠模式、信息密度和路径清晰度。

## 🤖 执行流程

### 第1步：确定游戏板类型

根据游戏需求选择游戏板类型：

**标准游戏板**（共享，多人使用）：
- Small: A5 → A4（小型游戏）
- Standard: A4 → A3（标准游戏，最常见）
- Large: A3 → A2（大型游戏）
- Extra Large: A2 → A1（超大游戏）

**玩家垫板**（个人使用）：
- Small: A6（参考卡）
- Standard: A5（个人面板）
- Large: A4（复杂垫板）

### 第2步：选择布局系统

根据游戏机制选择合适的布局：

**网格系统**：
- Hex Grid（六边形网格）- 适合地图拼接
- Square Grid（方格网格）- 适合区域划分
- Irregular Grid（不规则网格）- 适合区域地图

**区域系统**：
- Central Zone Design - 中央主区域
- Multi-Zone Layout - 多区域布局

### 第3步：设计信息层级

应用视觉优先级：

**Level 1（最重要）**：
- 游戏板标题/Logo
- 主游戏区域
- 关键计分轨道
- 回合顺序指示器

**Level 2（重要）**：
- 次要计分
- 资源轨道
- 行动空间
- 图例/说明

**Level 3（辅助）**：
- 装饰元素
- 背景细节
- 次要图标

### 第4步：处理折叠线

**关键规则**：
- 内容距离折叠线：15-20mm
- 关键内容距离折叠线：25mm+
- 四周出血：5mm
- 裁切安全区：15mm

### 第5步：验证信息密度

**观看距离指南**：

| 观看距离 | 最小元素 | 推荐 | 最大 |
|----------|----------|------|------|
| 30-50cm（垫板） | 3mm | 5-8mm | 15mm |
| 50-80cm（近距） | 5mm | 8-12mm | 20mm |
| 80-120cm（标准） | 8mm | 12-18mm | 30mm |
| 120cm+（大型板） | 12mm | 18-25mm | 40mm |

**字体比例**（垫板 vs 游戏板）：

| 元素 | 垫板 | 游戏板 |
|------|----------------|------------------|
| 标题 | 14-18pt | 24-36pt |
| 标题文字 | 10-12pt | 18-24pt |
| 正文 | 8-10pt | 14-18pt |
| 数字 | 10-14pt | 18-24pt |

### 第6步：设计路径系统

选择路径类型：

**Linear Path（线性路径）**：
- 用途：竞赛轨道、战役进程、科技树
- 特点：方向明确、选择有限、易于理解

**Branching Path（分支路径）**：
- 用途：冒险游戏、技能进阶、故事进程
- 特点：玩家选择、多条路线、部分回溯

**Network Path（网络路径）**：
- 用途：地图移动、贸易路线、连接游戏
- 特点：多重连接、开放式、策略规划

**Area Movement（区域移动）**：
- 用途：区域地图、领土控制、区域影响
- 特点：自由移动、基于区域、灵活

### 第7步：验证制作规范

**分辨率**：300 DPI @ 最终尺寸

**颜色模式**：CMYK

**出血**：5mm 四周

**安全区**：距裁切边缘 15mm

**文件格式**：PDF/X-4 推荐

## 🎯 快速决策指南

**选择游戏板尺寸**：
- 2-4玩家，标准组件 → A4 → A3
- 2-6玩家，大量组件 → A3 → A2
- 6+玩家，超大地图 → A2 → A1

**选择厚度**：
- 2面板 → 2mm
- 4面板 → 2-2.5mm
- 6面板+ → 2.5-3mm

**选择表面处理**：
- 预算有限 → 无涂层或哑光
- 标准游戏 → 哑光或光油
- 高级游戏 → 亚麻纹理
- 儿童游戏 → UV 涂层
