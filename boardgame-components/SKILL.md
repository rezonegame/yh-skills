# Board Game Components Design

桌游配件的完整设计指南，涵盖标记、指示物、玩家辅助、规则书和记分板。

## 🤖 执行流程

### 第1步：确定配件类型

**配件类别**：
- Tokens（标记）- 资源、货币、分数
- Markers（指示物）- 玩家位置、行动状态
- Player Aids（玩家辅助）- 参考卡、垫板
- Rulebooks（规则书）- 游戏规则、指南
- Score Pads（记分板）- 记录表格

### 第2步：设计标记和指示物

**标准尺寸**：
- 小型：15-20mm（资源标记）
- 中型：20-25mm（通用标记）
- 大型：25-30mm（玩家标记）

**设计要点**：
- 观看距离：30-80cm
- 最小图标：8mm
- 形状识别：清晰区分

### 第3步：设计玩家辅助

**辅助卡类型**：
- 参考卡：快速参考
- 玩家垫板：个人面板
- 规则摘要：核心规则

**布局原则**：
- 层级清晰
- 图文结合
- 易于理解

### 第4步：设计规则书

**规则书结构**：
1. 设置
2. 游戏目标
3. 游戏流程
4. 组件说明
5. 游戏结束

**排版指导**：
- 字体：10-12pt
- 行距：1.4-1.6
- 段落间距：6-10pt

### 第5步：设计记分板

**布局类型**：
- 线性轨道
- 表格式
- 区域式

**尺寸**：A5 或 A4
